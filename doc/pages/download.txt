
File: Getting remotestorage.js

Group: For development

Topic: Getting the source

Our sources are hosted on <github at http://github.com/remotestorage/remotestorage.js>.

You can clone them like this:

> git clone git://github.com/remotestorage/remotestorage.js

Or download a tarball:

> wget https://github.com/remotestorage/remotestorage.js/tarball/master


Group: Ready made builds

We provide different builds of remotestorage.js. This section describes what is what

Build Target: remotestorage.min.js

Minified build.

<download at http://remotestoragejs.com/release/0.7.3/remotestorage.min.js>

Build Target: remotestorage-debug.js

Not minified build. Use this if you're getting errors and want to be able to read the code.

<download at http://remotestoragejs.com/release/0.7.3/remotestorage-debug.js>

Build Target: remotestorage-node.js

Minified build for use in nodejs.

<download at http://remotestoragejs.com/release/0.7.3/remotestorage-node.min.js>

Build Target: remotestorage-node-debug.js

Debug build for use in nodejs.

<download at http://remotestoragejs.com/release/0.7.3/remotestorage-node-debug.js>
